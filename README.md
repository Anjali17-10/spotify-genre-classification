# Spotify Genre Classification 🎵

This project focuses on classifying Spotify tracks into different music genres using machine learning techniques.

## 📌 Project Overview
- Dataset: Spotify audio features dataset
- Task: Multi-class genre classification
- Model Used: Random Forest Classifier
- Evaluation Metrics: Accuracy, Precision, Recall, F1-score

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Results
- Achieved ~83% overall accuracy
- High performance on majority classes
- Demonstrates class imbalance challenges in multi-class classification

## 🚀 How to Run
1. Clone the repository
2. Install required libraries
3. Run `spotify.ipynb` in Jupyter Notebook

## 📁 Files
- `spotify.ipynb` – Complete data preprocessing, training, and evaluation pipeline
- `README.md` – Project documentation
